# 🤖 AI Voice Chatbot – CODSOFT Internship

This project is developed as part of the CODSOFT Artificial Intelligence Internship.

## 💡 Project Description
This is a College Assistant Chatbot with:
- Text chat
- Voice input (Speech Recognition)
- Voice output (Text to Speech)
- GUI using Tkinter

The chatbot answers basic college-related queries like:
- Courses
- Fees
- Placements
- Date & Time

## 🛠️ Technologies Used
- Python
- Tkinter
- SpeechRecognition
- pyttsx3
